A multi-\/platform voxel editor with smooth surface generation, supporting desktop and VR (Meta Quest 3).\hypertarget{md__r_e_a_d_m_e_autotoc_md576}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md576}

\begin{DoxyItemize}
\item {\bfseries{Multi-\/resolution voxel system}}\+: 10 size levels (1cm to 512cm)
\item {\bfseries{Dynamic workspace}}\+: Adjustable from 2m³ to 8m³
\item {\bfseries{Smooth surface generation}}\+: Using Dual Contouring algorithm
\item {\bfseries{Group management}}\+: Organize voxels into hierarchical groups
\item {\bfseries{Visual feedback}}\+: Green outline previews and face highlighting
\item {\bfseries{Command-\/line interface}}\+: Full-\/featured CLI with auto-\/completion
\item {\bfseries{Mouse interaction}}\+: Click-\/to-\/place voxels with visual feedback
\item {\bfseries{File formats}}\+: Custom binary format (.cvef) and STL export
